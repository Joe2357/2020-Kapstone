1. 블루투스 모듈과 아두이노 보드의 RX, TX는 반대로 설정할 것
	- 보드의 송신 -> 모듈의 수신, 모듈의 송신 -> 보드의 수신?

2. AT 명령어를 통한 기본 설정을 할 것
	- AT : 블루투스 모듈이 정상작동하는지 테스트
	- AT+NAME? : 이름 불러오기
	- AT+NAME[Para] : 이름 변경하기 ( 현재 JGM-Joe로 설정 )
	- AT+PASS? : 비밀번호 불러오기
	- AT+PASS[Para] : 비밀번호 변경하기 ( 현재 235711로 설정 )
	- AT+TYPE3 : 스마트폰과 모듈 간의 통신 커넥션 설정

3. 앱인벤터
	- 기본적으로 제공하는 Bluetooth 모듈이 아닌, BluetoothLE를 이용해야 정상 연결이 가능함
